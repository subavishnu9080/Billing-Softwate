Shop Billing Software

A simple and efficient Billing Software developed for my father's retail shop to streamline daily sales and billing operations. This application helps manage product details, generate bills, calculate totals automatically, and maintain transaction records with accuracy.

Features
Product management (Add, Update, Delete products)
Fast bill generation
Automatic calculation of subtotal, tax, and total amount
Customer invoice creation
Sales record management
User-friendly interface
Secure data storage using a database
Technologies Used
Python
Flask/Django
HTML, CSS, JavaScript
MySQL/SQLite
Bootstrap
Purpose

This project was built to solve real-world billing challenges in a small retail business by reducing manual calculations, improving billing speed, and maintaining organized sales records.
